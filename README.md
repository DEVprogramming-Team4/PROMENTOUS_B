# PROMENTOUS_B

NODEjS (EXPRESS ) 기반의 API ROUTER / MYSQL 쿼리 작업 진행

#   terminal 명령어

- vue 실행하는 터미널과는 별도의 터미널 열어서 PROMENTOUS_B 폴더로 cd 진입
- server 진입해서 npm i XXXXX 실행
- cd ./server
- npm i express
- npm i dotenv
- npm i mysql
- npm i express-session
- npm i cors
- node .\app_mysql.js or .\app.js ( port 3000 실행 )
